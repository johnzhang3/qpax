# Examples

Three worked applications that show how a differentiable QP layer slots
into a larger learning or control problem:

- **[Bilevel optimization](bilevel.md)** — outer objective differentiated
  through an inner QP.
- **[MPC autotuning](mpc-autotuning.md)** — learn the cost weights of a
  model-predictive controller end-to-end.
- **[Multi-agent safety filter learning](multi-agent-safety.md)** — joint
  optimisation of per-agent safety projections.
